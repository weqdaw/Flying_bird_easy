# Flappy Bird 小游戏

一个使用 C++ 和 EasyX 图形库开发的 Flappy Bird 风格的小游戏.

## 游戏介绍

这是一个经典的 Flappy Bird 类型游戏，玩家控制一只小鸟穿越管道障碍物。游戏具有简单的物理系统和碰撞检测。

### 游戏特性

- 简单的键盘控制（空格键使小鸟上升）
- 随机生成的管道障碍物
- 重力物理系统
- 碰撞检测
- 游戏结束动画
- 背景音乐

## 🛠️ 技术栈

- **编程语言**: C++
- **图形库**: EasyX (用于 Windows 平台的简单图形编程)
- **多媒体**: Windows 多媒体 API (背景音乐播放)

## 编译和运行

### 环境要求

- Windows 操作系统
- Visual Studio 或其他支持 EasyX 的 C++ 编译器
- EasyX 图形库

### 编译步骤

1. 安装 EasyX 图形库
2. 在支持 EasyX 的 IDE 中打开项目
3. 确保所有资源文件在正确路径
4. 编译并运行

## 游戏控制

- **空格键**: 使小鸟向上飞行
- 游戏目标: 尽可能长时间生存，穿越更多管道

## 注意事项

- 确保所有图片和音频文件与可执行文件在同一目录
- 游戏需要在 Windows 环境下运行
- 需要正确安装 EasyX 库才能编译
